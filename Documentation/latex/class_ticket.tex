\doxysubsection{Detailed Description}
The class keeps number of row, number of seat in the hall. It has a status that indicates wheter the ticket is free, booked or sold. Innitially, the status if free. The class also keeps a string, which represents a note in case the ticket was booked. It contains a code that is innitially empty and fills only when the ticket is sold.
