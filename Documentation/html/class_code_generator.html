Generates the next code.
Example: generate(5, 6, 1)
Last code = A8B3C5B
Next code = A5B6C1C
It increments the last letter. If it was Z, then it becomes A and the third letter increments. If it was Z, then it becomes A and the second letter increments and so on. Finnaly, sets the last code to the generated code.
